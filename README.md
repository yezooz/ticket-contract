# Planar Network - Travel Ticket Contract

ERC721 compliant Solidity contract for travel tickets.

## Use

The ABI files for the contract can be included into other projects using npm:

```
npm install --save @planar/wallet
```

## Build

```
npm install
npm run build
```

## Test

```
npm test
```
